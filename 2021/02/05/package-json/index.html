package.json

npm 에서 라이브러리들을 관리해 줄 수 있게 나의 node프로젝트의 버전, 이름과 필요한 라이브러리들을 명시해 놓은 명세서와 같다.
명세서에 적힌 라이브러리들을 npm install 을 통하여 설치할 수 있으며 save를 통하여 이곳에 필요한 라이브러리들을 추가하여
어떤 라이브러리가 이 프로젝트에 포함되어야 하는지 명시해 줄 수 있다.
